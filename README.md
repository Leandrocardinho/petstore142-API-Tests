# Petstore API Tests

Automação de testes de API do Swagger Petstore para estudo e portfólio QA.
Projeto desenvolvido para demonstrar habilidades em testes automatizados de APIs, boas práticas em Java e organização de testes.

## Tecnologias utilizadas

- Java 17
- Maven (gerenciamento de dependências)
- JUnit 5 (framework de testes)
- RestAssured (teste de APIs)
- JSON e CSV (arquivos de massa de dados)

**Escopo do Projeto e Funcionalidades Testadas**

O projeto realiza automação de testes de API do Swagger Petstore, com foco em:

- **Pets**
  - Criação, leitura, atualização e exclusão de pets
- **Usuários**
  - Cadastro, autenticação e gerenciamento de usuários
- **Pedidos**
  - Criação e rastreamento de pedidos de compra
- Validação de respostas da API (status code, body, headers)
- Uso de arquivos de massa (CSV/JSON) para testes parametrizados

- ## Como rodar os testes

1. Clone o repositório:
```bash
git clone https://github.com/Leandrocardinho/petstore142-API-Tests.git

Entre na pasta do projeto: cd petstore142-API-Tests

Execute os testes via Maven: mvn test

Certifique-se de ter o Java e Maven instalados no seu computador.
